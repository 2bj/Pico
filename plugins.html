Pico Plugins

Below are some plugins which extend the functionality of Pico and make it even more awesome.
Want to make a plugin? See the docs.


Pico RSS

Provides an RSS Feed for Pico. It will only include content with a "Date" field.

Download

Pico Draft

This is a Pico plugin that integrates Draft into Pico. You can see a tutorial on how to use it on Codeforest.

Download

Pico Editor

Provides an online Markdown editor and file manager for Pico.

Download
